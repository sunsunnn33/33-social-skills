# 33-social-cover

逐步收集本期视频信息 → 生成封面提示词 → 用户确认 → 直接生成封面图 → 裁剪多平台比例版本。

**重要：一次只问一个问题，等用户回答后再问下一个。**

---

## Language

用中文回复。文件路径和字段名保持英文。

---

## Step 0 — 读取品牌配置

检查以下路径是否存在 COVER.md：

```
{folder}/COVER.md          ← 项目级（优先）
~/.33-skills/COVER.md      ← 全局配置（fallback）
```

**有 COVER.md → 静默读取，进入 Step 1。**

**无 COVER.md → 进入初始化，完成后继续 Step 1：**

```
还没有封面品牌档案，先建立一个。回答几个问题：

1. 账号内容方向？（例：匹克球教学 / 健身训练 / 科技测评）
2. 固定信任徽章有哪些？每行一个，例如：
   🇨🇳前国家队
   🇨🇳中网协导师
3. 类别标签常用哪些？（例：新手入门、新手进阶、高阶挑战）
4. 常用品牌色有哪些？（颜色名即可，例：荧光绿、热血红、电紫）
```

收集完后写入 `~/.33-skills/COVER.md`（使用文末模板），继续 Step 1。

## Step 1 — 收集本期信息

逐一询问，一次只问一个问题：

---

**第1问：主标题**

```
这期封面的主标题是什么？
（通常是账号品牌词 + 本集技术主题，例：「匹克球 / 零基础教学」，两行）
```

---

**第2问：副标题**

```
副标题（本集具体内容）是什么？
（例：步伐 站位（脚步跟练））
```

---

**第3问：类别标签**

列出 COVER.md 中配置的标签，让用户选：

```
右上角用哪个标签？

1. 新手入门
2. 新手进阶
3. 高阶挑战
4. 自定义（请输入）
```

---

**第4问：底部金句**

```
底部 ✅ 金句是什么？
（一句话总结本期价值，例：步伐移动 保姆级拆分）
```

---

**第5问：顶部色块颜色**

列出 COVER.md 中配置的色系，让用户选：

```
顶部色块用哪个颜色？

1. 荧光绿（冲击力强，适合入门/基础技术）
2. 热血红（激烈感，适合对抗/挑战技术）
3. 电紫（神秘感，适合高阶/冷门技术）
4. 其他（请描述）
```

---

**第6问：动作参考照片**

```
请上传这期拍摄的运动动作照片。

可以上传 1-3 张，用于保持人物一致性和动作感。
照片会作为封面图的人物参考。
```

## Step 2 — 生成封面提示词

根据 COVER.md 视觉规范 + 用户本期输入，生成完整封面提示词：

---

**展示给用户确认：**

```
=== 封面提示词 ===

[根据以下规范和用户输入生成的完整提示词]

风格：中国短视频/小红书教学封面，高对比高密度，运动感强
比例：3:4 竖版

背景结构：
顶部约 38% 为纯色色块（{用户选择的颜色}），色块底部与运动人物照片
自然渐变融合，不要硬边。

人物（参考上传照片）：
参考图中的人物，真人全身或半身运动出镜，{用户描述的动作/照片中的动作}，
动作充满爆发力和运动感。人物位于画面中部，占画面 45-55%，
人物与背景自然融合，不需要抠图描边。

文字排版：
① 主标题（顶部色块内）：超大中文字
  第一行：「{主标题第一行}」
  第二行：「{主标题第二行}」
  字体：综艺感超粗黑体，竖向拉伸紧凑，白色填充 + 粗黑描边
  文字占满顶部区域宽度，有强烈视觉压迫感

② 右上角：黄绿色（#AAFF00）圆角矩形标签，黑色粗体文字「{类别标签}」

③ 中部左侧（人物旁）：两个白色胶囊形徽章：
  「{信任徽章1}」
  「{信任徽章2}」
  白色圆角矩形底色，黑色文字，轻微投影

④ 人物下方：中等大小白色中文「{副标题}」，粗黑描边

⑤ 底部：绿色方形勾选框 + 粗体白色中文「{底部金句}」，黑色描边

整体规范：
高对比度，强情绪，一眼可读，信息密度高
不要贴纸装饰，不要留白，不要遮挡人物脸部
所有文字不超出安全区

=================

这个提示词可以直接生成封面，确认吗？
（可以说"确认"，或告诉我哪里要修改）
```

如果用户要修改，调整对应部分，重新展示，直到用户确认。

---

## Step 3 — 生成封面图

用户确认后，直接按照封面提示词生成图片：

- 使用封面提示词作为图片生成指令
- 将用户上传的动作参考照片作为人物参考
- 生成 3:4 竖版封面图

生成完成后展示图片，询问：

```
封面图已生成。

需要裁剪其他平台比例吗？

1. 9:16（抖音竖版）
2. 16:9（YouTube / Bilibili 横版）
3. 两个都要
4. 不需要，完成
```

---

## Step 4 — 裁剪多平台比例（可选）

如果用户需要其他比例，对生成的封面图执行裁剪：

**3:4 → 9:16（上下补边，用品牌主色填充）：**
```bash
ffmpeg -i "{cover_3x4}" -vf "pad=iw:iw*16/9:0:(oh-ih)/2:color={COLOR_HEX}" "{output_9x16}" -y
```

**3:4 → 16:9（中心横向裁剪）：**
```bash
ffmpeg -i "{cover_3x4}" -vf "crop=iw:iw*9/16:0:(ih-oh)/2" "{output_16x9}" -y
```

裁剪完成后汇报：

```
✓ 封面生成完成

文件：
  cover-3x4.jpg   → 小红书（原始）
  cover-9x16.jpg  → 抖音
  cover-16x9.jpg  → YouTube / Bilibili

保存在：{folder}/
```
